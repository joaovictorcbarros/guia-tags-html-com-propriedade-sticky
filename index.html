Guia de Tags em <html> e seus exemplos com descrição usando a propriedade sticky
<html>
<html>
Exemplo: <html><head><title>Meu Site</title></head><body>Conteúdo aqui</body></html>
A tag <html> é a raiz de um documento HTML. Ela indica o início e o fim do documento HTML.
<head>
<head>
Exemplo: <head><title>Título da Página</title><meta charset="UTF-8"></head>
A tag <head> contém metadados sobre o documento, como o título, links para estilos e scripts.
<title>
<title>
Exemplo: <title>Meu Site</title>
A tag <title> define o título do documento, que aparece na aba do navegador.
<meta>
<meta>
Exemplo: <meta name="viewport" content="width=device-width, initial-scale=1">
A tag <meta> fornece metadados, como a configuração de viewport para responsividade.
<link>
<link>
Exemplo: <link rel="stylesheet" href="estilo.css">
A tag <link> é usada para vincular arquivos externos, como folhas de estilo CSS.
<script>
<script>
Exemplo: <script src="script.js"></script>
A tag <script> é usada para incluir scripts, como JavaScript, no documento.
<body>
<body>
Exemplo: <body><h1>Bem-vindo ao meu site!</h1><p>Este é um parágrafo.</p></body>
A tag <body> contém todo o conteúdo visível do documento, como texto, imagens e links.
<h1>
<h1>
Exemplo: <h1>Título Principal</h1>
A tag <h1> define o título principal da página, sendo a mais importante entre os cabeçalhos.
<h2>
<h2>
Exemplo: <h2>Título Secundário</h2>
A tag <h2> define um subtítulo, sendo a segunda mais importante entre os cabeçalhos.
<h3>
<h3>
Exemplo: <h3>Título Terciário</h3>
A tag <h3> define um subtítulo de nível inferior, sendo a terceira mais importante entre os cabeçalhos.
<p>
<p>
Exemplo: <p>Este é um parágrafo.</p>
A tag <p> define um parágrafo de texto.
<a>
<a>
Exemplo: <a href="https://www.exemplo.com">Link</a>
A tag <a> define um hiperlink que pode ser clicado para navegar para outra página ou site.
<img>
<img>
Exemplo: <img src="imagem.jpg" alt="Descrição da imagem">
A tag <img> é usada para inserir imagens no documento.
<ul>
<ul>
Exemplo: <ul><li>Item 1</li><li>Item 2</li></ul>
A tag <ul> define uma lista não ordenada, onde os itens são marcados com marcadores.
<ol>
<ol>
Exemplo: <ol><li>Item 1</li><li>Item 2</li></ol>
A tag <ol> define uma lista ordenada, onde os itens são numerados.
<li>
<li>
Exemplo: <li>Item 1</li>
A tag <li> define um item de lista, que pode ser usado em listas ordenadas ou não ordenadas.
<table>
<table>
Exemplo: <table><tr><th>Nome</th><th>Idade</th></tr><tr><td>João</td><td>30</td></tr></table>
A tag <table> define uma tabela, que pode conter linhas e células de dados.
<thead>
<thead>
Exemplo: <table><thead><tr><th>Nome</th><th>Idade</th></tr></thead><tbody><tr><td>João</td><td>30</td></tr></tbody></table>
A tag <thead> é usada para agrupar o cabeçalho de uma tabela, que contém as células de cabeçalho (<th>).
<th>
<th>
Exemplo: <table><tr><th>Produto</th><th>Preço</th></tr><tr><td>Maçã</td><td>R$ 2,00</td></tr></table>
A tag <th> define uma célula de cabeçalho em uma tabela. As células de cabeçalho são geralmente exibidas em negrito e centralizadas por padrão.
<tbody>
<tbody>
Exemplo: <table><thead><tr><th>Nome</th><th>Idade</th></tr></thead><tbody><tr><td>João</td><td>30</td></tr></tbody></table>
A tag <tbody> é usada para agrupar o corpo de uma tabela, que contém as células de dados (<td>).
<tr>
<tr>
Exemplo: <tr><td>Célula 1</td></tr>
A tag <tr> define uma linha em uma tabela.
<td>
<td>
Exemplo: <td>Célula 1</td>
A tag <td> define uma célula de dados em uma tabela.
<form>
<form>
Exemplo: <form action="/submit" method="post"><input type="text" name="nome" placeholder="Digite seu nome"></form>
A tag <form> é usada para criar um formulário que pode enviar dados para um servidor.
<input>
<input>
Exemplo: <input type="text" name="nome" placeholder="Digite seu nome">
A tag <input> é usada para criar campos de entrada em um formulário.
<textarea>
<textarea>
Exemplo: <textarea name="descricao">Texto aqui...</textarea>
A tag <textarea> é usada para criar uma área de texto onde o usuário pode inserir texto longo.
<select>
<select>
Exemplo: <select name="opcao"><option>Opção 1</option><option>Opção 2</option></select>
A tag <select> é usada para criar uma lista suspensa de opções.
<option>
<option>
Exemplo: <option value="1">Opção 1</option>
A tag <option> define uma opção em uma lista suspensa.
<button>
<button>
Exemplo: <button type="submit">Clique aqui</button>
A tag <button> é usada para criar um botão que pode ser clicado para executar uma ação.
<header>
<header>
Exemplo: <header>Cabeçalho do site</header>
A tag <header> define um cabeçalho para um documento ou seção.
<nav>
<nav>
Exemplo: <nav><a href="#home">Home</a></nav>
A tag <nav> é usada para definir uma seção de navegação em um documento.
<main>
<main>
Exemplo: <main>Conteúdo principal</main>
A tag <main> é usada para definir o conteúdo principal de um documento.
<section>
<section>
Exemplo: <section>Seção do site</section>
A tag <section> é usada para definir uma seção temática em um documento.
<article>
<article>
Exemplo: <article>Artigo do site</article>
A tag <article> é usada para definir um conteúdo independente que pode ser distribuído ou reutilizado.
<aside>
<aside>
Exemplo: <aside>Barra lateral</aside>
A tag <aside> é usada para definir conteúdo relacionado, como uma barra lateral.
<figure>
<figure>
Exemplo: <figure><img src="imagem.jpg" alt="Descrição"><figcaption>Legenda da imagem</figcaption></figure>
A tag <figure> é usada para encapsular conteúdo ilustrativo, como imagens e suas legendas.
<figcaption>
<figcaption>
Exemplo: <figcaption>Legenda da imagem</figcaption>
A tag <figcaption> é usada para fornecer uma legenda para o conteúdo dentro da tag <figure>.
<details>
<details>
Exemplo: <details><summary>Mais informações</summary>Conteúdo adicional</details>
A tag <details> é usada para criar um widget que pode ser aberto ou fechado, mostrando ou escondendo conteúdo adicional.
<summary>
<summary>
Exemplo: <summary>Resumo do site</summary>
A tag <summary> fornece um resumo ou título para o conteúdo dentro da tag <details>.
<footer>
<footer>
Exemplo: <footer>Rodapé do site</footer>
A tag <footer> é usada para definir o rodapé de um documento ou seção, geralmente contendo informações de copyright ou links.
